Leetcode 刷题报告 #1

2020年05月05日 by uprightclear

1.两数之和

给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。

你可以假设每种输入只会对应一个答案。但是，数组中同一个元素不能使用两遍。

示例:

给定 nums = [2, 7, 11, 15], target = 9

因为 nums[0] + nums[1] = 2 + 7 = 9 所以返回 [0, 1]

来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/two-sum 著作权归领扣网络所有。

题解(Java)

class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<>();
        for (int i = 0; i < nums.length; i++) {
            int complement = target - nums[i];
            if (map.containsKey(complement) && map.get(complement) != i) {
                return new int[] { i, map.get(complement) };
            }
            map.put(nums[i], i);
        }
        throw new IllegalArgumentException("No two sum solution");
    }
}

思路

运用HashMap哈希表来解决此题。

创建complement临时变量，依次遍历数组中的元素，求出target与当前元素的差值并储存在临时变量complement中，通过查找哈希表中是否含有与complement相等的key来寻找目标的指针，若不存在，则将当前遍历元素及其对应指针存储在哈希表中，以备之后的遍历查找。若遍历结束仍未找到对应结果则throw报错。

该代码返回值数组内元素无法实现升序，但不影响题目要求，可进行排序后再return。
