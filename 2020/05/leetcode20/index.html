136.只出现一次的数字

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

说明：

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

示例 1:

输入: [2,2,1] 输出: 1

示例 2:

输入: [4,1,2,1,2] 输出: 4

来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/single-number 著作权归领扣网络所有。

题解(Java)

class Solution {
    public int singleNumber(int[] nums) {
        for (int i=1 ; i<nums.length ; i++) {
            nums[0] ^= nums[i];
        }
        return nums[0];
    }
}

思路

使用异或运算，利用异或运算的性质：自己本身相异或为0，0与任何数字异或都等于它本身。

a⊕b⊕a=b⊕a⊕a=b⊕(a⊕a)=b⊕0=b
